# E-commerce Backend API

Backend REST API for an e-commerce system built with Spring Boot.

## Tech Stack
- Java
- Spring Boot
- Spring Security
- JWT Authentication
- Spring Data JPA
- Maven
- MySQL

## Features
- User authentication (JWT)
- Product management
- Category management
- Order system
- Shopping cart
- Product images

## Project Structure

src/main/java
- controller
- service
- repository
- entity
- configuration

src/main/resources
- application.yml

## Getting Started

### Clone project

git clone https://github.com/TonTuan11/ecommerce-website.git

### Run application

./mvnw spring-boot:run

Application runs at:

http://localhost:8080
